Project Description: This is a simple calculator that allows up to 8 characters in the input fields one and two. It features the four arithmetic operators, multiplication (*), division (/), addition (+), and subtraction (-). The radio buttons are used to control which input field will be utilized by the buttons on the calculator's panel. An error will appear if the user enters a non-numeral input, prompting the user to enter numbers instead.
