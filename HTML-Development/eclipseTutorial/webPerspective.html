Utilizing Web Perspective In Eclipse

First we start by changing our perspective to web view. If you are using eclipse configured for web applications,
this will be the default view. However, if eclipse is also being used for other languages then this will need to
be configured before you can begin working. Simply click on the "open perspective" tab.

Select the web perspective to change the layout of your IDE.

Through this perspective you can keep track of your elements, as well as seeing the child-parent relationships
Elements will be created in the sidebar view in real time, making working with hierarchies easier. In the
example we can see how the p element is a child of the body parent element.
