Eclipse Starter Tutorial

  1. Creating A New Project



    Start out by creating a new web project by selecting "file", then "new", then "static web project".
    This will create a new project for you to create your HTML and other file types in.

  2. Naming Your Project



    Name your project in the "Project Name" text field. Make sure to be clear in any names you use.
    In this tutorial, I have named my project "Example".
