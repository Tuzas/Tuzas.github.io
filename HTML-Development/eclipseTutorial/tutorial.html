Eclipse Starter Tutorial

  1. Creating A New Project



    Start out by creating a new web project by selecting "file", then "new", then "static web project".
    This will create a new project for you to create your HTML and other file types in.

  2. Naming Your Project



    Name your project in the "Project Name" text field. Make sure to be clear in any names you use.
    In this tutorial, I have named my project "Example". Click "Finish" to generate the project.

  3. Creating An HTML File



    Right Click on your project and then select "New", followed by "HTML File". This will open up another window
    prompting you to name your file. Make sure you name your files accurately, this helps to organize them.

  4. Naming Your HTML File



    Name your HTML file here, clicking on finish after you are done doing so.

  5. Created HTML File



    If everything was done correctly, your file should be displayed in the middle window, ready for work.
    Next, lets create a CSS File for our example project.

  6. Naming Your CSS File



    We start out the same way as creating an HTML File, by going to the file menu and selecting "new".

  7. Working With Different Files



    Here, as with the HTML file, we name our CSS file.

  8. Creating a CSS File



    As a result we have two files. We can switch between working on one or the other either by
    clicking on the file names on the left side of the screen or on the tabs at the top of the screen.
